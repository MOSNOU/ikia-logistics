1. مقدمه

جریان ارزش (Value Stream) مجموعه فعالیت‌هایی است که از یک نیاز یا فرصت آغاز شده و به خلق ارزش برای ذی‌نفعان منتهی می‌شود.

در iKIA هدف صرفاً دیجیتالی کردن فرآیندها نیست.

هدف ایجاد جریان‌های ارزش یکپارچه از کشف فرصت تا خلق درآمد و اعتماد است.

2. اصول طراحی جریان‌های ارزش

تمام Value Stream های iKIA بر اساس اصول زیر طراحی می‌شوند:

13. تعریف موفقیت

موفقیت زمانی حاصل می‌شود که تمامی جریان‌های ارزش بدون اصطکاک، دیجیتال، قابل اندازه‌گیری و مقیاس‌پذیر باشند.

14. نتیجه‌گیری

Value Streams ستون فقرات مدل کسب‌وکار iKIA هستند.

تمام قابلیت‌های محصول، فرآیندها، داده‌ها و فناوری‌ها باید در خدمت بهینه‌سازی این جریان‌های ارزش قرار گیرند.
