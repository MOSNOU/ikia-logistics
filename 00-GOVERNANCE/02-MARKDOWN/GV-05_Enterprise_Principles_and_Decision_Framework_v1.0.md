اصول سازمانی و چارچوب تصمیم‌گیری iKIA

Document Code: GV-05Version: 1.0Status: DraftClassification: Enterprise Governance Document

1. مقدمه

در طول عمر یک سازمان هزاران تصمیم اتخاذ می‌شود.

برخی از این تصمیم‌ها کوچک هستند.

برخی دیگر مسیر آینده سازمان را تغییر می‌دهند.

هدف این سند ایجاد یک چارچوب واحد برای تصمیم‌گیری در تمامی سطوح iKIA است.

4. اصل اول

اعتماد مهم‌تر از رشد است

اگر بین:

رشد سریع

و

اعتماد پایدار

تعارض ایجاد شود،

اعتماد انتخاب خواهد شد.

5. اصل دوم

ارزش مهم‌تر از درآمد است

درآمد نتیجه ارزش‌آفرینی است.

درآمدی که بدون ارزش ایجاد شود پایدار نخواهد بود.

6. اصل سوم

شبکه مهم‌تر از تراکنش است

هدف iKIA صرفاً انجام تراکنش نیست.

هدف ایجاد شبکه‌ای پایدار از همکاری‌ها است.

7. اصل چهارم

قابل حسابرسی

8. اصل پنجم

کاربر مالک داده خود است

iKIA امانت‌دار داده است.

نه مالک داده.

9. اصل ششم

امنیت پیش‌فرض است

امنیت نباید یک قابلیت اختیاری باشد.

امنیت باید بخشی از طراحی اولیه باشد.

10. اصل هفتم

سادگی برای کاربر

پیچیدگی باید در پشت صحنه مدیریت شود.

11. اصل هشتم

اتوماسیون قبل از نیروی انسانی

هر فرآیند باید ابتدا برای اتوماسیون طراحی شود.

12. اصل نهم

API First

تمام قابلیت‌ها باید قابلیت استفاده از طریق API را داشته باشند.

13. اصل دهم

AI Native

تمام قابلیت‌های جدید باید از منظر استفاده از هوش مصنوعی ارزیابی شوند.

14. اصل یازدهم

Cloud Ready

تمام اجزای پلتفرم باید برای مقیاس ملی و منطقه‌ای طراحی شوند.

15. اصل دوازدهم

Open Integration

پلتفرم نباید جزیره‌ای باشد.

باید قابلیت اتصال به اکوسیستم را داشته باشد.

16. اصل سیزدهم

Vendor Independence

وابستگی بلندمدت به یک فروشنده فناوری ممنوع است.

17. اصل چهاردهم

Evidence Based Decisions

تصمیمات باید مبتنی بر داده و شواهد باشند.

18. اصل پانزدهم

Long-Term Thinking

تصمیمات باید اثرات بلندمدت داشته باشند.

19. اصل شانزدهم

National Value Creation

هر تصمیم باید از منظر ارزش‌آفرینی ملی ارزیابی شود.

20. اصل هفدهم

Ecosystem First

منفعت اکوسیستم بر منفعت کوتاه‌مدت پلتفرم اولویت دارد.

21. اصل هجدهم

Neutral Platform

iKIA باید بی‌طرف باقی بماند.

22. اصل نوزدهم

Transparency by Design

شفافیت باید در طراحی سیستم لحاظ شود.

23. اصل بیستم

Auditability

تمام فعالیت‌ها باید قابل حسابرسی باشند.
